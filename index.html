CdsDbfit

DbFit is a database testing framework that supports easy test-driven development of your database code.

View project onGitHub

DbFit Enabled for the iSeries.

DbFit is a framework that allows us to blend SQL verifications with our REST response verifications when testing with RestFixture. We use RestFixture to issue the REST request and verify the XML response with XPATH expressions and very simple SQL queries. DbFit allows for more robust queries that are easier to write and to read. Using this approach, we focus on the REST response in CdsRestFixture, and we focus on verifying the client database using DbFit.
